# Student Score Analyzer

A beginner-friendly Python project that analyzes student marks from a CSV file and shows performance statistics.

---

## What this project does

- Reads student marks from a CSV file  
- Calculates subject-wise average, minimum, and maximum marks  
- Calculates total and average marks for each student  
- Identifies pass/fail count and topper  
- Creates a bar chart of subject-wise average marks  

---

## Technologies Used

- Python  
- Pandas  
- Matplotlib
